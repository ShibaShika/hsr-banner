# 崩壞：星穹鐵道 - 限定躍遷一覽表 (HSR Banner Tracker)

這是一個用來追蹤《崩壞：星穹鐵道》角色限定躍遷（卡池）歷史與復刻間隔的互動式表格網頁。

🔗 **線上即時預覽**：請至 GitHub Pages 設定後查看您的網頁專案網址。

---

## 📂 專案資料夾結構

本專案採用模組化結構，方便日後維護與更新：
* `index.html`：網頁主架構與核心渲染邏輯。
* `css/style.css`：網頁專屬樣式與配色定義。
* `js/characters.js`：角色資料庫（包含角色名稱、命途、屬性、復刻紀錄）。
* `js/patches.js`：遊戲版本與對應日期資料。
* `js/paths.js`：命途圖標連結與順序配置。
* `js/elements.js`：屬性分類配置。
* `updater.py`：自動化更新腳本。

---

## 💡 如何進行日常維護與更新？

如果您未來需要隨著遊戲更新來調整內容，可以按照以下方式操作：

### 1. 新增或修改角色
開啟 `js/characters.js` 檔案，在陣列中按照現有格式加入新角色：

    { name: "角色名稱", path: "命途", elem: "屬性", avatar: "", runs: ["版本1", "版本2"] }

### 2. 新增遊戲版本日期
開啟 `js/patches.js` 檔案，在陣列尾端加入新的版本與日期：

    {"patch": "4.5上", "date": "26/08/25"}
