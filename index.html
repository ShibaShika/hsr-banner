💡 站長維護指南 (如何更新此網頁)

這是一個純靜態且自動化排版的 HTML 網頁。您只需要將此檔案 index.html 放到 GitHub 儲存庫並開啟 GitHub Pages 即可發佈。

📝 未來要更新卡池時，請用文字編輯器打開此檔案，滑到最底部的 <script> 區塊：

  1. 新增版本：在 PATCHES 陣列最後面加上新版本字串 (例如 "4.4下")。
  2. 更新角色卡池：在 CHARACTERS 陣列中找到該角色，將剛才的版本字串加進他的 runs 陣列中。

※ 網頁載入時會自動為您重新計算所有角色的未復刻次數並標示漸層顏色，徹底告別臃腫且需手動算格子的 Excel！
